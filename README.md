# adrenaline
A Simple 3D Shooting Game 

https://user-images.githubusercontent.com/55603984/185768554-08497ba9-81bb-4a93-a200-18f143c03a36.mp4

## Requirements
* SDL2
* OPENGL >= 4.1

## How To Build And Run
```sh
git clone https://github.com/vmraa/adrenaline.git
cd adrenaline
mkdir build 
cd build
cmake .. && make
./adrenaline ..
```
